# <img src="logo.png" width="40" height="40"> Vansan Therm

**Vansan Therm** é uma solução inteligente de monitoramento térmico desenvolvida para **farmácias de manipulação**. O projeto utiliza a tecnologia IoT (Internet das Coisas) para garantir que insumos e medicamentos sejam armazenados dentro das normas rigorosas da **ANVISA**.

---

## 🚀 O Projeto
Este sistema foi desenvolvido como parte do **Empreenda Senac 2026**, com o objetivo de automatizar o controle de temperatura e umidade, reduzindo falhas humanas e perdas de estoque.

### Principais Funcionalidades:
* **Monitoramento em Tempo Real:** Leitura constante de temperatura e umidade via sensores.
* **Conformidade ANVISA:** Configurado para os padrões de 15-25°C e 40-70% de umidade.
* **Alertas Visuais:** Sistema de LEDs indicadores (Azul e Vermelho) para status do ambiente.
* **Interface Web:** Visualização dos dados através de uma página gerada pelo ESP32 ou GitHub Pages.

---

## 🛠️ Tecnologias Utilizadas
* **Hardware:** Microcontrolador ESP32.
* **Linguagem:** C++ (Arduino IDE) / Dart & Flutter (App de controle).
* **Front-end:** HTML5, CSS3 e JavaScript para o relatório de monitoramento.
* **Integração:** Webhooks e automação de notificações.

---

## 📋 Configuração de Hardware (Pinagem)
* **LED Vermelho (Alerta):** Pino D5
* **LED Azul (Status):** Pino D4
* **Sensor de Temperatura/Umidade:** Conectado via barramento de dados ao ESP32.
